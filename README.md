# 🤖 AI Code Reviewer

An AI-powered code review tool built with the **MERN Stack**. Paste your code, click **Review**, and get instant AI feedback on quality, bugs, performance, and best practices.

---

## ✨ Features

- 📝 **Live Code Editor** — Write or paste code with syntax highlighting
- 🤖 **AI-Powered Review** — Get instant feedback from Google Generative AI
- 🎨 **Clean UI** — Simple, responsive, and beginner-friendly interface
- ⚡ **Fast Results** — Markdown-formatted review output in seconds

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, TailwindCSS, PrismJS, React Markdown |
| Backend | Node.js, Express.js |
| AI | Google Generative AI (Gemini) |
